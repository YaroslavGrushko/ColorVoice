Privacy Policy of COLOR VOICE app

1. DANGEROUS/SENSITIVE ANDROID PERMISSIONS REQUESTED BY THIS APP

• Permission: RECORD_AUDIO

RECORD_AUDIO permission allows an application to record audio.

2. WHAT PURPOSE DANGEROUS/SENSITIVE ANDROID PERMISSIONS SERVE

We use RECORD_AUDIO permission ONLY to display user’s voice overtones energy as appropriate color in this application.

3. ANY OTHER USER DATA THIS APP COLLECT

This application does not collect any other user data.

4. SECURITY

User’s recording is saved into internal storage of device.
It means that:
  • File saved here is accessible by only Color Voice app.
  • When the user uninstalls Color Voice app, the system removes all Color Voice’s files from internal storage.
